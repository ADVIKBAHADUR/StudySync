# Project Tracker

Semester task dashboard with Google Sheets integration.

## Setup

1. Copy `config.example.js` to `config.js`
2. Edit `config.js` and add your credentials:
   - Google Sheet ID
   - Google API Key
   - Apps Script URL
3. Open `index.html` in a browser or deploy to GitHub Pages

**Note:** `config.js` is gitignored to keep your credentials private.
